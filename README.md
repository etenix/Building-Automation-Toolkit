# Building Automation Toolkit

## 概要
建築外壁の数量算出を自動化するデスクトップアプリケーション。  
従来手作業で行われていた外壁数量計算を、画像解析により効率化することを目的とする。

---

## 使用技術
- Python
- OpenCV（画像補正・前処理）
- YOLOv5（タイル検出）
- PyQt（GUI開発）

---

## 主な機能
- 画像補正による認識精度向上
- タイルの自動検出
- 外壁数量の自動算出
- GUIによる直感的操作

---

## 工夫点
- 照明条件の違いに対応するための画像前処理を実装
- 誤検出を抑えるための閾値調整と後処理ロジックを設計

---

## 担当
- 画像処理アルゴリズム設計
- YOLOモデルの統合・調整
- GUIアプリケーション開発
