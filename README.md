# 建築外壁解析および数量算出自動化ツールキット

## プロジェクト概要
本プロジェクトは、**協和建材株式会社**での実務に基づき開発した、建築設計支援ツールです。
現場写真からタイルの数量を自動算出し、設計データの管理およびAutoCAD/Revitへの連携を効率化します。

## 主な機能
- 画像補正エンジン: OpenCVを用いた透視変換により、斜めに撮影された写真から正対した画像を生成。
- AI数量算出: YOLOv5を用いたタイルの自動識別により、手動でのカウント作業を完全自動化。
- 直感的なGUI: PySide6(Qt)を採用し、現場担当者が容易にAI解析を実行できる環境を提供。

## 技術スタック
- Python: OpenCV, PyTorch, PySide6
- CAD連携: AutoCAD, Revit (設計支援業務)
- OS: Windows (業務環境)

## 導入成果
- 設計支援業務におけるデータ処理時間を大幅に削減。
- ヒューマンエラー（数え間違い）を排除し、材料発注の精度を向上。
